- Java および Spring Boot フレームワークを利用して、コア財務の拡張性を確保する高性能なバックエンドアプリケーションモジュールを概念化および保守。
- リアルタイムの電子収入印紙（e-Materai）処理を自動化するため、財務省（Kemenkeu）との安全な統合フレームワークを実行。
- 複雑なリレーショナルデータベースクエリを分析・最適化し、PostgreSQL および Microsoft SQL Server 内で戦略的なデータインデックス計画を確立。
- 自動化されたエンドツーエンドのデータパイプラインを構築・管理して運用統計を収集し、JasperReports を利用して正確な財務報告書を作成。
- Nginx をリバースプロキシとして構成し、オンプレミスサーバーの展開アーキテクチャを管理。SSL/TLS 暗号化と OAuth2 により環境を保護。
- COVID-19 救済イニシアチブである**Pemulihan Ekonomi Nasional (PEN)**、**Agunan Bank Online**プラットフォーム、および**Surety Bond Online**システムのバックエンドソリューションを設計。
