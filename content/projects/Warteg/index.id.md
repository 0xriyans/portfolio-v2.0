FoodRestaurants adalah aplikasi web yang dibangun menggunakan Vue.js untuk melakukan browse dan pemesanan makanan dari berbagai restoran. Aplikasi ini memiliki clean & simple interface untuk mengeksplorasi menu restoran dan melakukan place orders, sehingga memberikan seamless experience bagi para penggunanya.
