Sebagai Backend Developer untuk [Penjaminan KMK PEN](https://askrindo.co.id/penjaminan-kmk-pen) di Asuransi Kredit Indonesia, saya mengembangkan dan memelihara backend systems yang mendukung layanan penjaminan kredit. Saya membangun dan mengoptimalkan API menggunakan Java (Spring Boot) untuk memfasilitasi secure loan application processing, guarantee issuance, dan real-time data tracking. Peran saya juga melibatkan integrasi dengan sistem eksternal untuk risk assessments, guna memastikan data processing yang efisien dan scalable bagi klien maupun internal users.
