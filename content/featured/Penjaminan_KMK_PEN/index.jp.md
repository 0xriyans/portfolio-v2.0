Asuransi Kredit Indonesia の [Penjaminan KMK PEN](https://askrindo.co.id/penjaminan-kmk-pen) におけるバックエンド開発者として、ローン保証サービスを支えるシステムの開発と保守を行いました。Java (Spring Boot) を使用して API を構築・最適化し、安全なローン申請処理、保証の発行、およびリアルタイムのデータ追跡を実現しました。また、リスク評価のための外部システムとの統合にも携わり、クライアントと社内ユーザーの双方にとって効率的でスケーラブルなデータ処理を確保しました。
